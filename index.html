HOW TO MASTER GOOGLE SEARCH ?




If you want to find the right answer,
You need to ask the right question!





FOUND A WEBSITE YOU LIKE?
Use the keyword "related:" to find similar websites.
Example: related:Technology.com




SEARCH WITHIN A WEBSITE?
Use the keyword "search:" to search within a website.
Example: search:Technology.com Articles




SEARCH THE EXACT PHRASE?
Use the double quotation "" for finding the exact phrase.
Example: "Remote IT Developer Jobs"




WANT TO EXCLUDE A KEYWORD?
Use the keyword " - " to remove a certain phrase or word from your search.
Example: Best Programming language -HTML




HAVE YOU FORGOT A WORD?
Use the keyword " * " for missing words.
Example: How to use float in Django version *?




LOOKING FOR A DEFINITION?
Use the keyword "define:" to find a definition.
Example: define:Python
